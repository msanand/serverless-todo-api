CSE 204A API Key

This page provides the documentation for getting an API Key to use for assignments in WUSTL CSE 204A.

Please type your WUSTL email address in the field below and click "submit". A unique API key will be generated that you can use to submit and retrieve data. This API Key is unique to you. If you lose your API key, you will also lose any data associated with it. However, you can generate a new API key at any time with the same name and email address, to start over with new data.
